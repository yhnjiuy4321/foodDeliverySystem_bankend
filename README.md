# 餐點外送平台管理系統 (後端)

這個專案是餐點外送平台的後端部分，運用 Spring Boot 進行開發，並使用 Java 來處理與資料庫的串接。

## 系統架構
後端使用 **Spring Boot** 搭配 **MariaDB** 作為資料庫。提供 RESTful API 供前端與資料庫進行交互。
